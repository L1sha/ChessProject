# ChessProject

CLI based Chess game!


- 현재 말이 위치한 칸을 입력해 움직일 말을 선택하고, 움직일 칸을 입력
- 움직일 말을 선택하였을 때 선택한 말이 움직일 수 있는 칸들을 알려줌
- White / Black 2가지의 Perspective 제공
- Restart 기능 구현
- 앙파상, 캐슬링, 프로모션 구현
- Last move 표시 기능 구현
- 항복 기능 구현
